How to Use

Basic Workflow

  1. Navigate to any image gallery website
  2. Click the StepFour icon to open the side panel
  3. Check if gallery was auto-detected
  4. Select pagination method (or use auto-detect)
  5. Click "Start Pagination"
  6. Watch images collect in real-time
  7. Export to CSV or download all images
